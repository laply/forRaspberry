-- sensing info 
각각 6개의 센서 

temp/humid 
= (온도/습도) - 8~12 초 마다 한번 씩 보낸다.  

String 데이터 

형식
"Temperature : 온도값 C"
"Humidty : 습도값 %"

fire / shock / IR 
= (불꽃감지 / 충격 / 근접감지) - 일이 발생했을때 값을 출력

String 데이터 
형식 
"1" or "0"

clear 
= (버튼) - 물리적 클릭시 작동

String 데이터
형식 
"0" or "1"
